  • 사용프로그램

    HTML / CSS / JavaScript / Photoshop / Xd / JSP / Mysql / Visual Studio Code / NotePad

  • 프로젝트 소개

    처음으로 규모가 있는 웹 프로젝트를 2명이서 작업한 결과물입니다. 제가 디자인과 웹페이지 파트를 맡아 HTML + CSS를 이용해 기본을 다지고 부트스트랩과 자바스크립트를 이용하여 반응형 웹 페이지를 만들었습니다.


    또한 MySQL 이용해 DB를 만들어 사용자들이 구매한 내역 및 취향을 DB에 저장하여 사용자들에게 상품랭크기능이나 별점 후기시스템을 이용해 제품을 추천해주고 KAKAO API를 이용해 결제까지 완료하고 확인 할 수 있도록 하여 최대한 실제 웹페이지와 같은 완성도 높은 페이지를 만들었습니다.


    현재는 페이지의 주력기능인 커스터마이즈 꽃다발에 DB의 저장된 상품들을 토대로 추천기능을 접목시키고 있으며 테블릿, 모바일에서도 웹 페이지를 사용 할 수 있도록 개발중입니다
